# Connect Four (Четыре в ряд)

Классическая стратегическая игра для двух игроков, реализованная на C# (.NET) с использованием Windows Forms. Проект демонстрирует разделение ответственности между бизнес-логикой и пользовательским интерфейсом.

## 🚀 Особенности
- **Архитектура**: Бизнес-логика вынесена в отдельную библиотеку `Connect4.Core`, что позволяет легко тестировать код и портировать его на другие платформы.
- **Адаптивный интерфейс**: Игровое поле автоматически масштабируется при изменении размера окна.
- **Таймер хода**: Ограничение времени на принятие решения (по умолчанию 15 секунд).
- **Покрытие тестами**: Основная логика покрыта модульными тестами xUnit.

## 🛠 Технологии
- .NET 8 / C# 12
- Windows Forms (UI)
- xUnit (Unit Testing)
- GDI+ (Custom Rendering)

## 📂 Структура проекта
- `Connect4.Core`: Ядро игры (правила, проверка победы, таймер).
- `Connect4.UI`: Пользовательский интерфейс (главное меню, игровая форма, кастомная отрисовка).
- `Connect4.Tests`: Набор модульных тестов для проверки логики.

## 🎮 Как запустить
1. Откройте файл `ConnectFour.sln` в Visual Studio 2022 или JetBrains Rider.
2. Установите проект `Connect4.UI` как запускаемый (Startup Project).
3. Нажмите `F5` для сборки и запуска.
